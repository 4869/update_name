update_name
===========

##What is it?

Change Your Name on Twitter by reply

##How to Use

###Require

* ruby
* bundle

###Use 

```
$ git clone https://github.com/sh4869/update_name.git
$ cd update_name
$ bundle install
$ bundle exec ruby update_all.rb
```
